# Cookbook

## Описание

Сервис для создания и распространения рецептов блюд

## Функциональность

- Создание новых рецептов
- Создания перечня необходимых продуктов и количество
- Пространство для описания поэтапного приготовления
- Возможность настройки, создания заметок
